# School Management System

Simple Console App for managing Teachers and Students using C#.

## Features

### Teacher
- Add Teacher
- Show Teachers
- Update / Delete (basic CRUD)

### Student
- Add Student
- Show Students
- Update / Delete (basic CRUD)

## Tech
- C#
- .NET Console
- OOP
- List
- Interface

## Structure

```text
Models
Services
Program.cs
```

dotnet run
